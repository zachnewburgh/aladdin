# Aladdin

This gem shows an ASCII image of Disney's Aladdin in the command line.

## Installation

In your terminal:

```
$ gem install aladdin
```

## Usage

To use the gem, type the following your terminal:

```
$ irb

> Aladdin::CLI.new.call

```
